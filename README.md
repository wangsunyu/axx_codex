# 安小信 Flutter 项目

一个面向 iOS、Android、鸿蒙的 Flutter 项目基座，提供共享应用壳层、标准移动端宿主工程，以及鸿蒙接入占位结构。

## 快速开始

```bash
flutter pub get
flutter run
```

## 项目结构

- `lib/`：共享 Flutter / Dart 代码
- `assets/`：应用资源与占位配置
- `ios/`：iOS 宿主工程
- `android/`：Android 宿主工程
- `harmony/`：HarmonyOS 宿主接入脚手架与说明
- `docs/`：开发、运行与平台接入文档
